## User Journeys

### Journey 1: First-Time Visitor (Awareness → Consideration)

**Persona**: Sarah Mitchell, ranch owner looking for renovation

**Path**:
1. **Arrival**: Lands on home page via Google search ("luxury interior design West Texas")
2. **First Impression**: Sees hero image of ranch project, reads headline "Design That Feels Like Home"
3. **Exploration**: Scrolls to Philosophy section, reads about West Texas ranch aesthetic
4. **Validation**: Clicks through to Work page, filters by "Ranch & Estate"
5. **Connection**: Views project details, sees before/after of similar ranch renovation
6. **Trust**: Reads About section, learns about principal designer and process
7. **Consideration**: Returns to home page, reviews Services overview
8. **Action**: Clicks Contact, fills out consultation request form

**Duration**: 8-12 minutes
**Pages Viewed**: 5-7 pages
**Conversion**: Consultation request form

### Journey 2: Referral Visitor (Consideration → Conversion)

**Persona**: Michael Chen, referred by architect friend

**Path**:
1. **Arrival**: Lands on home page via direct link from architect's website
2. **Validation**: Immediately clicks Work to see portfolio quality
3. **Exploration**: Views 3-4 residential projects, reads project descriptions
4. **Connection**: Clicks About to understand design philosophy
5. **Process**: Reviews Process section to understand working style
6. **Services**: Reviews Services page to understand offerings and pricing
7. **Social Proof**: Reads testimonials from similar clients
8. **Action**: Clicks Contact, fills out form with project details

**Duration**: 5-8 minutes
**Pages Viewed**: 4-6 pages
**Conversion**: Consultation request form

### Journey 3: Real Estate Professional (Partnership Inquiry)

**Persona**: Elena Rodriguez, luxury realtor

**Path**:
1. **Arrival**: Lands on home page via search ("luxury home staging West Texas")
2. **Targeted**: Clicks Services, navigates to Home Staging
3. **Evaluation**: Reviews staging services, pricing, and process
4. **Validation**: Views Staging Projects portfolio, reviews before/after examples
5. **Trust**: Reads About section, understands team and expertise
6. **Information**: Reviews FAQ section for common questions
7. **Action**: Clicks Contact, selects "Real Estate Partnership" inquiry type

**Duration**: 6-10 minutes
**Pages Viewed**: 4-5 pages
**Conversion**: Partnership inquiry form

### Journey 4: Return Visitor (Engagement → Action)

**Persona**: Previous website visitor, now ready to act

**Path**:
1. **Arrival**: Returns to site via bookmark or direct URL
2. **Recall**: Recognizes brand, feels familiar with design
3. **Targeted**: Navigates directly to Contact page
4. **Review**: Quickly reviews contact information and service areas
5. **Action**: Fills out consultation request form

**Duration**: 2-3 minutes
**Pages Viewed**: 1-2 pages
**Conversion**: Consultation request form

## CTA Strategy

### Primary CTAs

**"Start Your Project" Button**
- **Placement**: Hero section, Services page, About page
- **Destination**: Contact page with pre-selected "Consultation" inquiry type
- **Copy**: "Start Your Project"
- **Style**: Primary button (accent color background, white text)
- **Priority**: Highest visibility

**"View Our Work" Button**
- **Placement**: Hero section, Home page
- **Destination**: Work page (all projects)
- **Copy**: "View Our Work"
- **Style**: Secondary button (transparent background, accent border)
- **Priority**: High visibility

**"Schedule a Consultation"** Button
- **Placement**: Services page, About page
- **Destination**: Contact page with pre-selected "Consultation" inquiry type
- **Copy**: "Schedule a Consultation"
- **Style**: Primary button
- **Priority**: High visibility

### Secondary CTAs

**"Learn More" Links**
- **Placement**: Throughout content sections
- **Destination**: Relevant detail pages
- **Copy**: "Learn More" or context-specific (e.g., "About Our Process")

- **Priority**: Medium visibility

**"View Project" Links**
- **Placement**: Portfolio cards, featured projects
- **Destination**: Individual project pages

- **Priority**: Medium visibility

**"Contact Us" Links**

- **Priority**: Medium visibility

## Conversion Funnels

### Funnel 1: Consultation Request (Primary)

**Stages**:
1. **Awareness**: User lands on site (any page)
2. **Interest**: User explores portfolio and services
3. **Consideration**: User reviews About/Process pages
4. **Intent**: User clicks CTA to Contact page
5. **Action**: User completes consultation request form
6. **Conversion**: Form submitted, follow-up initiated

**Drop-off Points**:
- After portfolio view (if projects don't resonate)
- After services review (if pricing unclear)
- At contact form (if form too long or complex)

**Optimization**:
- Strong portfolio with diverse project types
- Clear service packages and pricing ranges
- Simple contact form (3-5 fields maximum)

### Funnel 2: Project Inquiry (Secondary)

**Stages**:
1. **Awareness**: User lands on Work page
2. **Interest**: User views specific project
3. **Consideration**: User reviews project details and related projects
4. **Intent**: User clicks "Inquire About This Project" CTA
5. **Action**: User completes project inquiry form
6. **Conversion**: Form submitted, project-specific follow-up

**Drop-off Points**:
- After project view (if project doesn't match needs)
- At inquiry form (if form asks for too much information)

**Optimization**:
- Clear project categorization and filtering
- Simple inquiry form (name, email, project reference, message)

### Funnel 3: Partnership Inquiry (Tertiary)

**Stages**:
1. **Awareness**: Real estate professional lands on site
2. **Interest**: User reviews staging services
3. **Consideration**: User reviews staging portfolio and process
4. **Intent**: User clicks "Partner With Us" CTA
5. **Action**: User completes partnership inquiry form
6. **Conversion**: Form submitted, partnership follow-up

**Drop-off Points**:
- After services review (if pricing unclear)
- At partnership form (if form too complex)

**Optimization**:
- Clear staging service packages and pricing
- Simple partnership form (company, name, email, message)

## Navigation Behavior

### Desktop Navigation
- **Position**: Fixed top navigation bar
- **Visibility**: Always visible, semi-transparent on scroll
- **Items**: Work, Services, About, Contact
- **Logo**: Left side, links to home
- **CTA**: Right side, "Start Your Project" button
- **Hover**: Subtle underline animation on links
- **Mobile**: Hides on mobile, replaced by hamburger menu

### Mobile Navigation
- **Position**: Fixed top navigation bar with hamburger menu
- **Behavior**: Hamburger menu opens full-screen overlay
- **Overlay**: Full-screen menu with large touch targets
- **Items**: Home, Work, Services, About, Contact
- **Animation**: Slide in from right, smooth transition
- **Close**: X button in top right corner
- **Background**: Semi-transparent dark overlay

### Footer Navigation
- **Position**: Bottom of every page
- **Columns**: 4 columns (Navigation, Services, Contact, Social)
- **Links**: Work, Services, About, Contact, Privacy, Terms
- **Social**: Instagram, LinkedIn icons
- **Background**: Dark neutral color
- **Text**: Light neutral color

### Breadcrumb Navigation
- **Position**: Top of project pages, below navigation
- **Style**: Small text, gray color, links separated by ">"
- **Behavior**: Clickable links to parent pages
- **Home**: Always links to home page
- **Current**: Non-clickable, indicates current page

## Search Engine Optimization Structure

### URL Structure
- Clean, descriptive URLs
- Hyphen-separated words
- Lowercase only
- No unnecessary parameters

### Meta Information
- Unique title tags for each page
- Meta descriptions for each page
- Open Graph tags for social sharing
- Schema markup for local business

### Heading Structure
- Single H1 per page
- Logical heading hierarchy (H1 → H2 → H3 → H4)
- Keywords in headings where natural
- No skipped heading levels

### Content Structure
- Semantic HTML5 elements
- Alt text for all images
- Descriptive link text
- Sufficient content depth

## Accessibility Structure

### Keyboard Navigation
- Logical tab order
- Skip to content link
- Focus indicators
- No keyboard traps

### Screen Reader Structure
- ARIA labels where needed
- Landmark regions
- Heading hierarchy
- Alt text for images

### Visual Structure
- Sufficient color contrast
- Text resizable up to 200%
- No content on hover only
- Clear focus states
